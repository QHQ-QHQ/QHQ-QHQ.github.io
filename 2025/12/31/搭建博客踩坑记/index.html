搭建博客踩坑记

第一次自己搭建网站，虽然遇到了很多 404 错误，但最后都解决了！

遇到的坑

  1. GitHub 仓库名必须和用户名一样。
  2. Windows 下不能直接运行脚本，需要改权限。
  3. Butterfly 主题需要安装插件。

总结

感觉自己离程序员又近了一步！
